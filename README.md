# Hostel Management System

A complete command-line hostel management system with student, room, fee, attendance, visitor, and complaint management.

## Features

- Student registration, update, removal, search and listing
- Room creation, update, deletion, assignment, release and listing
- Fee payment recording and due balance tracking
- Attendance records by date
- Visitor registration and checkout
- Complaint logging and status updates
- Summary reports and CSV export
- Persistent storage with `hostel_data.json`

## Usage

Run the system from the project folder:

```bash
cd "c:\Users\Mahesh Pawar\Desktop\PROJECTS\HOSTEL_MANAGEMENT_SYSTEM"
python main.py
```

## Data Files

- `hostel_data.json`: stores all hostel records
- `students_export.csv`, `rooms_export.csv`, `complaints_export.csv`, `visitors_export.csv`: created when exporting data
